## English
This repository brings three programs for controlling a drone with bi-directional ESCs (running in one-direction mode), using an ESP32 (in progress) or Arduino for setup, calibration, and control. There is also a PCB branch that implements the project on a printed circuit board.

## Português (BR)
Este repositório traz três programas para o controle de um drone com ESCs bidirecionais (operando em modo de uma direção), utilizando ESP32 (em desenvolvimento) ou Arduino para configuração inicial, calibração e controle. Há também uma versão (branch PCB) para o projeto feito em uma placa de circuito impresso.

## 中文
此仓库提供三个程序，用于控制使用双向 ESC（以单向模式运行）的无人机，可选择 ESP32（开发中）或 Arduino 进行初始配置、校准和控制。项目还提供了一个 PCB 分支，将方案实现到印刷电路板上。
