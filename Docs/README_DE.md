## Deutsch
Dieses Repository stellt drei Programme zur Steuerung einer Drohne mit bidirektionalen ESCs (im Einrichtungsbetrieb einseitig laufend) bereit. Als Plattform kommen ein ESP32 (in Entwicklung) oder ein Arduino für Konfiguration, Kalibrierung und Flugsteuerung zum Einsatz.
